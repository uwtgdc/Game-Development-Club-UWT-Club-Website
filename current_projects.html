    Ongoing Projects

    Welcome to the current projects page of the Game Development Club @ UWT! Here you can learn about projects that are currently happening in the club!

    If you want to join any of these projects, please send a message into their respective channels in the Game Dev @ UWT Discord Server!

    Dystopian Space Huskies ~ An Original Game Made in Godot

    A husky is trying to fight to rescue its human from the dystopian powers-that-be on his celestial body.

    When the Project Started: October 2024

    Software Used in the Project: Godot and GitHub; Programming Language: C#

    Getting Saucy! ~ An Original Game Made in Unity

    A 2D platformer with mouse aiming shooting mechanics with pixel art and a level structure where you play as a chicken named Saucy.

    Main objective is to escape the fridge!

    When the Project Started: October 2024

    Software Used in the Project: Unity and GitHub; Programming Lanaguage: C#
